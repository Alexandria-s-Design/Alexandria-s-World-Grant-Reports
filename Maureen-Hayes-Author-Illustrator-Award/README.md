## Award Overview

**Program:** Maureen Hayes Author/Illustrator Award
**Funding:** Up to $4,000
**Purpose:** For libraries to host a nationally recognized children's author/illustrator visit
**Sponsor:** Simon & Schuster, administered by ALSC/ALA

---

## What Award Covers

- **Honorarium** for author/illustrator visit
- **Travel expenses** for author/illustrator to visit library
- **Books** for the library (up to $4,000 total)

---

## Eligibility

**Who Can Apply:**
- Public or school libraries
- ALA/ALSC membership may be required

**What's Needed:**
- Plan to host a nationally recognized children's author or illustrator
- Event for children or teens
- Library as host institution
