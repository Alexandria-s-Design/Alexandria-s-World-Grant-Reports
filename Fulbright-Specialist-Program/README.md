## Grant Overview

**Program:** Fulbright Specialist Program
**Duration:** 2-6 weeks (minimum 14 days, maximum 42 days)
**Funding:** Covers airfare, lodging, meals, visa, daily honorarium
**Eligibility:** U.S. citizens with significant expertise in supported fields
**Application Process:** Apply to Roster anytime; host institutions request specialists

---

## What the Program Offers

- **Short-term grants** for project-based exchanges abroad
- **Comprehensive coverage:** Airfare, lodging, meals, visa, daily honorarium
- **Flexible duration:** 2-6 weeks depending on project scope
- **Host cost-share:** Host institutions may need to cost-share some expenses

## Application Process

### Step 1: Apply to Roster (Ongoing)

- Submit application anytime through fulbrightspecialist.worldlearning.org
- Peer-reviewed approval required
- Approved specialists remain on Roster for 3 years
- No deadline for roster applications

### Step 2: Host Request

- Host institutions (universities, NGOs, organizations) request specialists
- Matches occur via U.S. Embassies or Fulbright Commissions
- Projects include: seminars, workshops, curriculum development, lecturing, needs assessment

### Step 3: Project Approval

- Host submits project proposal to Fulbright
- Specialist confirms availability and interest
- Project approved and scheduled
