# Alexandria's World Grant Reports

## Overview

This repository contains grant research, proposal drafts, and submission materials for the **Alexandria's World** children's book series project.

**Project Summary:** Alexandria's World is a culturally responsive children's book series centering an African American toddler experiencing diverse cultures through authentic journeys aligned with the National History Educational Standards (NHES) for culture education. Each book includes companion educational modules on cultural preservation, peace education, and being an upstander.
